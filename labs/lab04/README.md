# Lab 04 – RSS & Atom Feeds

## Description
This lab includes valid RSS 2.0 and Atom 1.0 XML feeds that publish updates for my ITWS-1100 labs.

## Files
- `rss.xml` – RSS 2.0 feed
- `atom.xml` – Atom 1.0 feed
- `rss.css` – CSS styling via `<?xml-stylesheet?>`

## Validation
Both feeds are well-formed XML and validate against the RSS 2.0 and Atom 1.0 specifications using the W3C Feed Validator.
